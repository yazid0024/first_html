# Web Portfolio

This project is a personal web portfolio designed to showcase projects and skills. It includes various components and styles to create an engaging user experience.

## Project Structure

```
web-portfolio
├── src
│   ├── index.html          # Main HTML document for the portfolio
│   ├── css
│   │   └── styles.css      # Styles for the portfolio
│   ├── js
│   │   └── main.js         # JavaScript for interactivity
│   ├── components
│   │   ├── header.html     # Header component with navigation
│   │   └── footer.html     # Footer component with copyright info
│   └── data
│       └── projects.json    # JSON data for projects
├── assets
│   └── fonts               # Directory for font files
├── package.json            # npm configuration file
├── .gitignore              # Files and directories to ignore by Git
└── README.md               # Project documentation
```

## Features

- Responsive design that adapts to different screen sizes.
- Interactive elements powered by JavaScript.
- Clean and modern UI with custom styles.
- Easy navigation through the header component.
- Dynamic project loading from JSON data.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd web-portfolio
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Open `src/index.html` in a web browser to view the portfolio.

## Usage

- Modify the `projects.json` file to update the project details displayed in the portfolio.
- Customize styles in `css/styles.css` to change the appearance of the portfolio.
- Add interactivity through the `js/main.js` file.
